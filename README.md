## FOREST-PREDICTION-HACKATHON
This is a hackathon project where we need to find the total estimated forest area and degraded land under Uttarakhand state and find the correct percentage value.
This project implements a LSTM machine learning model to predict forest-related values for the Uttarakhand region.
<br>
The workflow includes data preprocessing, feature analysis, model training, and prediction. The trained model analyzes the input parameters like the satellite images of uttarakhand region and predicts the output value, demonstrating how data-driven approaches can support environmental monitoring and decision making.
<br>
It shows the correct value of how much land is under forest area in uttarakhand and how much it is actually degaraded land. This model also represents deforestation and soil erosion of uttarkhand.
